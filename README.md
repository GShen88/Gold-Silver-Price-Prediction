# Gold-Silver-Price-Prediction

Dataset is based on the gold&silver price from Jan 2019 to Jun 2019 from https://www.investing.com/commodities/gold-historical-data and https://www.investing.com/commodities/silver-historical-data .

Analyzed stationarity & randomness of the time series data by visualization and Augmented Dickey-Fuller test.

Built, trained and fine-tuned the non-seasonal ARIMA model to predict future trend and forecast gold and silver price.

What to improve: coding style could be more concise; did not explain the offset very well
